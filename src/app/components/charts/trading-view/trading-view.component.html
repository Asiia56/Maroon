Tradingview

Where the world charts, chats and trades markets..

It's a supercharged super-charting platform and social network for traders and investors

How to use

TradingView is a charting platform used by traders and investors worldwide. TradingView widget is a free widget can be used in Ract, Angular and Vue. There is a angular-tradingview-widget is already built in github. Install failed, because angular-tradingview-widget is using Angular 9, not supporting Angular 13 yet. For build own tradingview-widget the source files in github repository was used
