Alpha Vantage

Major provider of historical and real-time stock APIs along with cryptocurrency, Forex, ETF, mutual funds data feeds

Each free API key allows up to 500 API calls per day by default.

How to use

API calls are made using any web-enabled client (e.g. a web browser) to make an HTTP GET request to an appropriate URL.
